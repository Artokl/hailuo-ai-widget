# Hailuo AI Video Generator Widget

Interactive high-conversion widget prototype built for Overchat AI / Pandaverse.

## Key Features 
- **Engagement-First UX:** Integrated a 3D Tilt effect and smooth Framer Motion transitions to create a premium, "living" interface.
- **AI Simulation Terminal:** Real-time log streaming during the generation process to reduce perceived wait time and increase user trust.
- **Conversion-Focused Result Screen:** Implemented a "Blur Reveal" effect and teased Pro features (4K, Music, Extra Duration) to drive registration intent.
- **Product-Driven UI:** Clean, modern "Hailuo" branding with custom-built selectors (replacing standard browser inputs).

## Tech Stack
- **React.js**
- **Tailwind CSS** (for rapid styling)
- **Framer Motion** (for advanced animations)
- **Cloudflare Workers** (Proxy architecture prepared)

## Integration
The widget is designed to be easily embedded into Webflow or any landing page via iframe or a custom script loader.
